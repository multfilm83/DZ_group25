Первое ДЗ (Gitbash) группа 25 

1) Посмотреть где я - pwd
2) Создать папку - mkdir "название папки" (пример: mkdir papka_1)
3) Зайти в папку - cd "название папки" (пример: cd papka_1)
4) Создать 3 папки -  mkdir "название папок" (пример: mkdir dir_1 dir_2 dir_3) 
5) Зайти в любоую папку - cd "название папки" (пример: cd dir_2)
6) Создать 5 файлов (3 txt, 2 json) - ДВА ВАРИАНТА: 
Вариант_1: touch "названия файлов с расширением через пробел" (пример touch 1.txt 2.txt 3.txt 4.json 5.json )
Вариант_2: > "названия файлов с расширением через пробел" (пример > 1.txt 2.txt 3.txt 4.json 5.json )
7) Создать 3 папки - mkdir "название папок" (пример: mkdir dir_4 dir_5 dir_6)
8. Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл - vim "название файла" (пример vim 1.txt)
10) + написать туда что-нибудь, любой текст. - нажать клавишу "i", набрать текст
11) + сохранить и выйти. - нажать клавишу "ESC", поставить двоеточие ":", набрать wq
12) Выйти из папки на уровень выше - cd ..
13) переместить любые 2 файла, которые вы создали, в любую другую папку. - mv 1.txt 2.txt dir_4/
14) скопировать любые 2 файла, которые вы создали, в любую другую папку. - cp {3.txt,4.json} dir_4/
15) Найти файл по имени - find -type f -name "filename" пример: find -type f -name 5.json
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает. - tail -f "filename" пример: - tail -f 1.txt
17) вывести несколько первых строк из текстового файла - head -n2 "filename" пример: head -n2 1.txt
18) вывести несколько последних строк из текстового файла - tail -n2 "filename" пример: tail -n2 1.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает. 
less "filename" пример less 1.txt, режим просмотра файла. Для выхода из режима просмотра нажать "q"
20) вывести дату и время - date
